# 将主窗口改为自定义无边框圆角壳

## 本次目标
- 让设置主窗口不再依赖系统标题栏，改成更明显的圆角无边框外观，并保留窗口拖拽与基本窗口操作。

## 核心改动
- 更新 `src-tauri/tauri.conf.json`，将主窗口改为 `decorations: false`，并启用 `transparent: true` 与 `shadow: true`。
- 新增 `src-ui/lib/tauri/window.ts`，统一封装主窗口的最小化、切换大小和收起到托盘操作。
- 更新 `src-ui/features/settings/SettingsWindow.tsx`，新增自定义拖拽头部和窗口按钮，并将设置页外层改为透明无边框壳层布局。
- 更新 `src-ui/app/App.tsx` 与 `src-ui/styles/index.css`，按窗口类型设置 `body` 背景策略，避免透明窗口外侧被默认背景填满。
- 更新 `TASK.md`，记录并收口本次任务状态。

## 修改文件
- `src-tauri/tauri.conf.json`
- `src-ui/lib/tauri/window.ts`
- `src-ui/app/App.tsx`
- `src-ui/styles/index.css`
- `src-ui/features/settings/SettingsWindow.tsx`
- `TASK.md`

## 校验情况
- 已执行 `npm run build`
- 已执行 `npx tauri build --debug`
- 已确认新的 debug 安装包已生成

## 风险或遗留项
- 本次主要验证了调试构建与安装包生成，尚未额外产出新的 release 包。
- 无边框窗口在不同 Windows 版本上的阴影和贴边缩放手感可能略有差异；如果你安装后觉得窗口边缘拖拽手感不够好，可以再补专门的 resize 热区优化。
