# 2026-04-18 Avalonia 首个可运行版本

## 本次目标

- 搭建 MiniNotifier 的首个 Avalonia 可运行版本，并验证 Native AOT 发布链路。

## 核心改动

- 新增 `src/MiniNotifier.Avalonia/` 工程，使用 `Avalonia 12.0.1 + CommunityToolkit.Mvvm + Generic Host`。
- 复用现有配置持久化、随机文案、自启动、鼠标活跃度与提醒调度相关共享代码。
- 实现 Avalonia 主设置窗口、提醒弹窗、托盘菜单和窗口管理。
- 在项目文件中加入 AOT 友好设置：`CompiledBindings`、`IsAotCompatible`、条件 `PublishAot/PublishTrimmed`。

## 修改文件

- `MiniNotifier.slnx`
- `src/MiniNotifier.Avalonia/`

## 校验情况

- 已执行 `dotnet build src/MiniNotifier.Avalonia/MiniNotifier.Avalonia.csproj`
- 已执行 `dotnet publish src/MiniNotifier.Avalonia/MiniNotifier.Avalonia.csproj -c Release -r win-x64`
- 已对发布后的 `MiniNotifier.Avalonia.exe` 做短时启动验证，进程可正常拉起

## 风险或遗留项

- Native AOT 发布阶段仍存在 `MicroCom.Runtime` 的 `IL3053` 警告，来源于依赖链，当前不阻塞发布，但后续需要持续关注。
- 目前只验证了进程可启动，尚未做完整的 UI 交互 smoke test。
