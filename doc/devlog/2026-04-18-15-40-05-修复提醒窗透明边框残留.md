# 修复提醒窗透明边框残留

## 本次目标
- 收紧提醒窗的窗口裁剪与页面壳层，修复透明无边框提醒窗仍出现异常边线、外层框感和边缘残留的问题。

## 核心改动
- 更新 `src-tauri/tauri.conf.json`，关闭提醒窗的系统阴影，减少透明窗口外侧矩形阴影残留。
- 更新 `src-ui/features/settings/ReminderWindow.tsx`，让提醒卡片更贴合窗口区域，统一卡片高度与圆角半径，并把底部操作区稳定压到卡片底部。
- 更新 `src-ui/styles/index.css`，为提醒窗根节点增加 `overflow: hidden`，避免透明窗口外层泄露出额外边缘。
- 更新 `src-tauri/src/lib.rs`，将提醒窗 Windows 侧裁剪半径统一为 `26`，并去掉额外的 `+1` 区域扩展，减少边缘错位。
- 更新 `TASK.md`，记录并收口本次问题修复任务。
- 执行 `npx tauri build --debug`，重新生成可测试安装包。
