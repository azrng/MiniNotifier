# 2026-03-28 设置窗口闪退与 Logo 修复

## 本次目标

- 修复托盘“打开设置”后窗口闪一下即消失的问题
- 调整应用 Logo 视觉，使其更贴合喝水提醒主题

## 核心改动

- 将设置数据初始化提前到应用启动阶段，避免首次打开设置窗口时再触发初始化隐藏逻辑
- 删除主窗口 `Loaded` 时立即 `Hide()` 的行为，修复“打开设置”闪退 / 闪隐问题
- 为窗口管理增加 Dispatcher 调度，避免托盘交互线程和 UI 线程发生潜在冲突
- 将原来的字母 Logo 改为水滴图形 Logo

- `src/MiniNotifier/App.xaml.cs`
