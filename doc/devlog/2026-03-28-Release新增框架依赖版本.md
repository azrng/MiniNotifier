# 2026-03-28 Release 新增框架依赖版本

## 本次目标
- 在现有 `x64`、`x86` 自包含发布基础上，额外提供一个依赖目标机现有 .NET 框架的版本

## 核心改动
- 调整 GitHub Actions 打包工作流，新增 `framework-dependent` 发布通道
- Release 产物改为三份：
  - `MiniNotifier-win-x64.zip`
  - `MiniNotifier-win-x86.zip`
  - `MiniNotifier-framework-dependent.zip`
- 框架依赖版采用多文件 ZIP，避免 WPF 在框架依赖单文件场景下出现异常膨胀和额外原生 DLL 问题

## 修改文件
- `.github/workflows/build.yml`
- `TASK.md`

## 校验情况
- 执行 `dotnet publish .\src\MiniNotifier\MiniNotifier.csproj -c Release --self-contained false -o .\artifacts\gha-check\framework-dependent -p:PublishSingleFile=false -p:DebugType=embedded -p:PublishReadyToRun=false -p:PublishTrimmed=false`
- 本地验证框架依赖版发布成功，总体积约 `8.5MB`

## 风险或遗留项
- 框架依赖版需要目标机器预先安装 `.NET Desktop Runtime 10`
- 当前已完成本地发布命令验证，仍建议首次推送后检查一次 GitHub Actions 的远端产物
