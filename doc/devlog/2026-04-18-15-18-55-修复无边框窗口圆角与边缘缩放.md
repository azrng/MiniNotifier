# 修复无边框窗口圆角与边缘缩放

## 本次目标
- 让无边框主窗口真正呈现圆角外轮廓，并恢复边缘拖拽缩放能力。

## 核心改动
- 更新 `src-tauri/src/lib.rs`，在 Windows 下为主窗口和提醒窗口同步设置 DWM 圆角偏好，并使用 `CreateRoundRectRgn + SetWindowRgn` 把窗口实际裁成圆角区域。
- 在窗口 `Resized / ScaleFactorChanged` 事件中重新应用圆角区域，避免缩放后又回到方角。
- 更新 `src-ui/lib/tauri/window.ts`，新增窗口边缘缩放封装。
- 更新 `src-ui/features/settings/SettingsWindow.tsx`，在主窗口四边和四角加入无边框缩放热区，支持鼠标拖拽调整窗口大小。
- 更新 `src-tauri/Cargo.toml`，补充 `Win32_Graphics_Gdi` 特性。
- 更新 `TASK.md`，记录并收口本次任务状态。
