# 2026-03-28 真实持久化与随机提醒文案

## 本次目标
- 将喝水提醒设置从 mock 切换为真实 JSON 持久化
- 接入 Windows 当前用户开机自启动
- 为提醒弹窗加入按时间段随机切换的内置文案，并移除当前时间显示

## 核心改动
- 新增 `JsonHydrationSettingsRepository`，配置文件落到 `%LocalAppData%/MiniNotifier/hydration-settings.json`
- 新增 `RegistryAutoStartService`，通过 `HKCU\\Software\\Microsoft\\Windows\\CurrentVersion\\Run` 控制开机自启动
- 新增 `HydrationSettingsService`，统一处理加载、保存、暂停、提醒记录与状态回填
- 新增 `ReminderMessageService`，内置多套晨间、午间、下午、傍晚、深夜和暂停预览文案
- 提醒弹窗去掉“当前时间”字段，改为随机标题 + 随机正文
- 设置窗口保存逻辑改为真实保存，并补充异常处理与提示
