# 2026-03-28 根据鼠标点击推断工作状态

## 本次目标
- 获取全局鼠标点击次数
- 通过最近一段时间的点击活跃度推断工作状态，并让提醒文案跟随变化

## 核心改动
- 新增 `MouseActivityService`，使用 Windows 全局低级鼠标钩子统计最近 1 分钟和 5 分钟点击次数
- 新增工作状态模型，将点击频率映射为 `DeepFocus / SteadyFlow / ActiveHandling / RapidFire`
- 在提醒文案中加入基于点击活跃度的上下文表达
- 启动时自动初始化鼠标活跃度服务，失败时只记录日志，不影响应用继续运行
