# 提醒弹窗卡顿优化

## 本次目标

优化喝水提醒弹窗出现时的短暂停顿，减少提醒触发瞬间对 UI 线程的集中占用。

## 核心改动

- 为提醒弹窗服务增加启动后空闲预热，提前创建并完成一次弹窗模板和布局初始化。
- 将弹窗定位中的同步 `UpdateLayout()` 移出实际弹框路径，改为布局完成后异步修正位置。
- 保留原有提醒展示、自动关闭和下次提醒时间记录逻辑。
