# 为窗口启用 Windows 圆角边框

## 本次目标
- 让 Tauri 设置窗口使用 Windows 原生圆角边框效果，弱化当前偏硬的直角标题栏外观。

## 核心改动
- 更新 `src-tauri/Cargo.toml`，补充 `Win32_Graphics_Dwm` 特性以便调用 DWM 窗口属性接口。
- 更新 `src-tauri/src/lib.rs`，在应用启动阶段为 `main` 与 `reminder` 窗口设置 `DWMWA_WINDOW_CORNER_PREFERENCE = DWMWCP_ROUND`。
- 更新 `TASK.md`，记录并收口本次任务状态。

## 修改文件
- `src-tauri/Cargo.toml`
- `src-tauri/src/lib.rs`
- `TASK.md`

## 校验情况
- 已执行 `cargo check --manifest-path src-tauri/Cargo.toml`
- 已重新读取关键改动片段，确认圆角设置在应用启动阶段生效

## 风险或遗留项
- 该圆角偏好主要在 Windows 11 上生效；若系统版本较低，可能不会看到明显变化。
- 本次未重新构建安装包或进行人工桌面 smoke test；如需确认最终视觉，仍建议重新运行或重新打包后验证。
