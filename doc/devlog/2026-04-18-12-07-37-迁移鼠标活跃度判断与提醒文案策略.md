# 迁移鼠标活跃度判断与提醒文案策略

## 本次目标
- 在 Tauri 侧补齐全局鼠标活跃度判断与工作状态推断。
- 让提醒文案根据时间段、工作节奏和星期上下文组合生成。

## 核心改动
- 新增 Rust 侧 `mouse_activity_service`，通过 Windows 低级鼠标钩子统计最近 1 分钟与 5 分钟点击量。
- 新增 `Unavailable / DeepFocus / SteadyFlow / ActiveHandling / RapidFire` 工作状态推断，并提供前端可消费的快照命令。
- 扩展提醒文案生成逻辑，按晨间 / 午间 / 下午 / 傍晚 / 深夜选择标题、标题文案、开场句，并叠加工作节奏与星期上下文。
- 设置页新增“工作节奏”卡片，展示当前推断状态和最近点击统计。
